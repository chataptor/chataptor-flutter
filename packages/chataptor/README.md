# chataptor

Pure-Dart core client for [Chataptor](https://chataptor.com) — real-time multilingual customer support chat.

> This is the **headless core**. For drop-in Flutter widgets, use [`chataptor_flutter`](../chataptor_flutter/).

## Install

```yaml
dependencies:
  chataptor: ^0.1.0
```

## Usage

See the [getting started guide](../../docs/guides/getting-started.md) or the [quickstart example](../../examples/quickstart/).
